# Agile-Development-co-pilot-and-version-control-system-use
The deployment of Scrum played a role in the creation of the Music Playlist and Recommendation Application that enabled the project to provide features iteratively and in an incremental fashion in a series of sprint cycles. Scrum was selected since it is adaptable, cooperative and able to meet the dynamic demands in an organized manner.
Definition of Ready (DoR)
Tasks were deemed to be ready when:
● The requirements were well defined.
● Criteria of acceptance were identified.
● Work could be effectively completed within a sprint.
Definition of Done (DoD)
It was determined that the tasks were complete when:
● Code was implemented and operational.
● Tests have been completed successfully.
● Alterations were made and added to the repository.
● The features were presented in the sprint review.
